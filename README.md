Sistema de Registro de Notas

Projeto desenvolvido em Python utilizando:
- Tkinter
- SQLite
- CRUD

Funcionalidades
- Cadastro de alunos
- Busca de alunos
- Atualização de dados
- Exclusão de alunos
- Listagem de alunos

Tecnologias utilizadas
- Python
- Tkinter
- SQLite3

Como executar
1. Abra o projeto no VSCode
2. Execute o arquivo interface.py
